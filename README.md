💼 Portfólio — Carrinho de Compras (Estilo Shopee)
📌 Descrição do Projeto

O projeto consiste no desenvolvimento da lógica de um carrinho de compras inspirado em plataformas de e-commerce. A aplicação permite gerenciar produtos de forma dinâmica, incluindo nome, preço unitário, quantidade e cálculo automático do valor total.

O foco principal foi a construção de uma estrutura organizada, modular e escalável, aplicando conceitos fundamentais de programação orientada a objetos.
🎯 Objetivo

Desenvolver um sistema de carrinho de compras funcional que simule operações reais de um e-commerce, como adicionar produtos, remover itens, calcular subtotal e total, além de implementar uma lista de desejos (wishlist).
🧠 Processo Criativo

O desenvolvimento foi baseado na separação de responsabilidades. A lógica foi dividida em módulos:

Item: responsável pelos dados e cálculo individual de cada produto
Cart: responsável por gerenciar o carrinho e aplicar regras de negócio
App: responsável por executar e testar as funcionalidades

Essa abordagem facilita manutenção, reutilização e expansão do sistema.
⚙️ Funcionalidades
Adicionar produtos ao carrinho
Remover produtos por nome ou índice
Calcular subtotal dos itens
Calcular valor total
Exibir carrinho no console
Gerenciar lista de desejos (wishlist)
🧱 Arquitetura
Estrutura modular com separação por arquivos
Uso de classes para organização da lógica
Código escalável e de fácil entendimento
🛠️ Tecnologias Utilizadas
JavaScript (ES6+)
Programação Orientada a Objetos (POO)
Módulos JS
📈 Conclusão
Este projeto demonstra a aplicação prática de conceitos fundamentais de programação, com foco em organização, reutilização de código e construção de lógica escalável, servindo como base para sistemas mais complexos de e-commerce.
